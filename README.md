# Wheels in Bali: E-Commerce & Vehicle Rental Front-End Website

This repository contains the front-end architecture, user interface components, and responsive layout assets developed for **"Wheels in Bali"**, a vehicle rental and e-commerce platform web system.

## Technology Stack & Specifications
* **Languages:** HTML5, CSS3, Vanilla JavaScript (DOM Manipulation)
* **Design Implementation:** Structured grid and flexbox layout frameworks ensuring seamless navigation across user flows.
* **Scope:** Dynamic client-side web interfaces tailored for business operational scenarios.

## Engineering Highlights
* **Clean Code Structure:** Styled using specialized, standalone CSS modular layers to separate formatting from document structure.
* **Interactive Architecture:** Configured interactive front-end mechanics (login verification templates, choice selection pathways) to support structured user routing.
